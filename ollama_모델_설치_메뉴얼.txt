Gemma3:4B 설치 방법 (Windows / macOS)

🔹 Windows에서 Gemma3 설치

1.  Ollama 설치

    -   https://ollama.com/download 에서 Windows 설치파일 다운로드 후
        설치

2.  PowerShell 또는 CMD 실행

3.  Gemma3 4B Pull 실행

        ollama pull gemma3:4b

4.  테스트 실행

        ollama run gemma3:4b

------------------------------------------------------------------------

🔹 macOS에서 Gemma3 설치

1.  Ollama 설치

    -   https://ollama.com/download 에서 macOS(.dmg) 버전 다운로드 후
        설치

2.  터미널 실행

3.  Gemma3 4B Pull 실행

        ollama pull gemma3:4b

4.  테스트 실행

        ollama run gemma3:4b

------------------------------------------------------------------------
